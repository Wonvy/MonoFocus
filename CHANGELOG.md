# 更新日志 / Changelog

所有重要的版本变更都会记录在此文件中。

## [1.1.0] - 2025-11-17

### 新增 / Added
- 🎨 更新应用图标，支持所有平台
- 🌍 多语言安装包支持（NSIS）
- 📦 中文 MSI 安装包
- 🔧 NSIS 安装包图标配置

### 优化 / Improved
- 优化安装包语言选择体验
- 改进系统托盘多语言支持

## [1.0.0] - 2025-11-17

### 新增 / Added
- ✨ 核心护眼功能：自动在非活动显示器上显示半透明遮罩
- 🖥️ 多显示器支持：自动检测所有连接的显示器
- 🎚️ 透明度调节：支持 0-100% 透明度设置
- 🎬 动画效果：支持无动画/快速/中等/慢速四种切换速度
- 🌍 多语言支持：简体中文、英语、日语、法语、德语、西班牙语
- ⚙️ 系统托盘：快速启用/禁用护眼模式
- 🚀 开机自启：支持系统启动时自动运行
- 💾 配置保存：自动保存所有用户设置

### 界面 / UI
- 🎨 极简黑白扁平化设计
- 📊 可视化显示器布局展示
- 🔢 显示器编号和分辨率显示
- 📏 自适应窗口布局，保持显示器比例

### 技术 / Technical
- 🦀 Rust + Tauri 构建，高性能低资源占用
- 🪟 Windows 平台完整支持
- 🔄 实时鼠标位置追踪
- 🎯 点击穿透遮罩窗口
- 💻 跨平台架构设计

---

## 版本号说明

版本号格式：`主版本号.次版本号.修订号`

- **主版本号**：重大架构变更或不兼容的 API 修改
- **次版本号**：新增功能或重要改进
- **修订号**：问题修复和小优化
